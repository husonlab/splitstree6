# SplitsTree 2.9.0

## New features

- SplitsTree6 releases are now made available via GitHub.
- Help -> Online User Manual... opens the user manual in a web browser
- SplitsTree6 tools are now shipped in a separate tools bundle. Note that the tools bundle
  contains tools/splitstree6, which also launches the program.

## Bug fixes

- Several minor bug fixes
- Message window now opens

## Known issues

- rotate and flip trees and networks do not always work correctly, this will be reimplemented soon.
